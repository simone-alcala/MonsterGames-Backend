# Getting started with Monster Games

## `.env SETUP`

- ### `MONGO_URI`

  - String de conexão com o banco de dados mongo
  - mongodb+srv://USUÁRIO:SENHA@cluster0.xpsue.mongodb.net/BANCO_DE_DADOS?retryWrites=true&w=majority

- ### `DATABASE`

  - Nome do banco de dados

- ### `PORT`

  - Porta de inicialização do serviço Node.js

- ### `JWT_KEY`

  - Chave de criptografia

- ### `EMAIL`

  - Endereço para envio de emails

- ### `EMAIL_PWD`

  - Senha do email para envio de emails

- ### `EMAIL_HOST`

  - Servidor SMTP para envio de emails

- ### `EMAIL_PORT`
  - Porta do servidor para envio de emails
